A) HTML Structure - top

If you check the HTML code of any layout you will find that each row is highlighted with comments showing its start and end, so you can easily delete or duplicate any of them, here is a similar color formation to what you get if you open any layout in Firefox and choose View Page Source, part of the code looks like that:



Tip: It's not necessary that source code will be shown with such colors in your HTML/Text editor! If it's shown in black just search for <!-- that's how all comments start.

You can see the green comments explaining where content starts and ends, if you decided to add a row or more just copy what's between code comments and paste it! For example you want to duplicate the text with picture on the left in the layout #1, copy from the code comment <!-- Start of text with picture on the left --> till <!-- End of text with picture on the left --> and paste it after <!-- End of text with picture on the left --> with this easy copy and paste method you can insert extra rows or delete them if you want, every row is created in separate table so it doesn't affect any other row whenever you duplicate/delete it.

Tip: You can mix layouts and create your own, but remember to copy and paste rows within the same color folder, so you won't end with different colored rows.

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

D) MailChimp Support - top

Please, follow these steps to import any Template to MailChimp:

1. Go to http://www.mailchimp.com and login.
2. Simply go to My Templates.
3. Click 'Import Template' and choose your Template from the desktop.

Be sure to have all your images hosted on your own server. For example: 'http://www.yourwebsite.com/images/logo.jpg'
