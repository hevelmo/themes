2.2 - Anthem HTML

Basic Anthem html is very easy to set up.


<body>
	<header class="main custom-scroll">
		<div class="header-scrollarea">
		We will set up header in the logo and navigation section to follow.
		</div>
	</header>
	<div class="trigger">
		We will set up trigger in the logo and navigation section to follow.
		<div class="trigger-box">
		</div>
		<div class="side-logo">
		</div>
		<div class="back-to-top">
		</div>
	</div>
	<!-- Main content block - all page content -->
	<div class="content custom-scroll">
		<div class="blocker">
		</div>
		<div id="scrollarea">
			<div class="action step-right">
			page content goes here
			
			<!-- for pages with multiple sections e.g home page or about page etc
				we place each section inside class="content-block" like below -->
				
				<div class="content-block">
					section content goes here.
				</div>
			
			<!-- you can have four defined classes of white, black, purple and grey -->
				<div class="white content-block">
					section content goes here. background is white color.
				</div>
			
			<!-- when using class=black add class=dark with it , which will apply lighter color styles on dark backgrounds.
			e.g white font color on black background-->
				<div class="black dark content-block">
					section content goes here. background is black color font is white color.
				</div>
			
			<!-- In case of grids, e.g portfolios, notes, equals blocks
				portfolio sections are explained later on this page.-->			
				<ul class="grid">	
				</ul>	
			<!-- you will then add either of the classes , class=times-three, class=times-four or class=times-two -->		
				<ul class="grid times-three">	
					three li's in one row, and so on.
				</ul>
				
			</div>
		</div>
	</div>
</body>

Html structure explanation.

class=content - All html content of page goes inside this class.

class=custom-scroll - Used to display that a custom scrollbar will be used.

class=blocker - Class becomes active when header is active. This places a black tint on page content and disables page scroll.

class=scrollarea - Adds a custom scrollbar.

class=action - The animation action to take when side header becomes active.

class=step-right - Page slides to right on active header. Other options include class="perspective", class="step-back", class="skew" i-e class="action skew"


Changing primary color

The easiest way to change the primary color is to open style.css in a text editor such as notepad++ and replace the value rgb(177,99,163) to your choice of color.

3 - RETINA SUPPORT

It is very easy to use retina images with anthem. Simply save dual copies of your images. Named image.jpg and image@2x.jpg . In your html simply add gumby-retina to your image tag , retina images will be loaded for Hi-res displays.


<img src="image.jpg" alt="image-alt" gumby-retina/>
4 - LOGO AND NAVIGATION

Anthem Side uses a side header and navigation. The header structure is explained below.


<header class="main custom-scroll">
	<div id="header-scrollarea">
		<div class="logo">
			<img src="logo-image-source" alt="logo" gumby-retina>
		</div>
		<!-- navigation -->
		<nav class="main-nav">
			<ul class="site-nav">
				<li><a href="" class="toggle" gumby-trigger="#id-of-child-drawer">Menu Item<span>Menu item description</span></a>
					<ul id="id-of-child-drawer" class="drawer">
						<li><a href="link-to-page.html">Child Link</a></li>
					</ul>
				</li>
				<!-- link without children -->
				<li><a href="link-to-page">Menu Item 2<span>Menu item description</span></a></li>
			</ul>
		</nav>
	</div>
</header>

Header structure explanation.

class=main - Main header that will display logo and navigation.

class=custom-scroll - Used to display that a custom scrollbar will be used.

class=logo - Place your logo inside this div.

nav class=main-nav - Primary site navigation.

ul class=site-nav - UL site navigation.

li > a class=toggle - Toggle class is added to toggle between active or inactive when child navigation open and closes

gumby-trigger=#id-of-child-drawer - We use gumby-trigger to target which child navigation should open when parent link is clicked. gumby-trigger must be same as the id of the child ul you want to target.

li > ul class=drawer - drawer class is added to hide content till the drawer is active.

Ajax homepage structure explanation.

class=open-slide - On click it will set body overflow to hidden. This class is not required if you are not using Ajax.

! Please note single project files for ajax portfolio must reside in projects folder. All ajax links must be set up in a similar way. Begining with #!projects/name-of-your-single-file.html

How to add portfolio filters to ajax page.

Adding portfolio filters is very easy. First add class=filter to class="grid alt" . This will initiate the isoptope filter function. Then you will have to add the below code for portfolio filter options, which is also found in folio-grid and folio-grid-alt. Next add the filter classes to each LI item as li class="grid-item filter-optiona filter-optionb" and so on.


<div id="options">
	<ul id="filters" class="option-set">
		<li><a href="#" data-filter="*" class="selected">Show All</a></li>
		<li><a href="#" data-filter=".filter-optiona">Option A</a></li>
		<li><a href="#" data-filter=".filter-optionb">Option B</a></li>
		<li><a href="#" data-filter=".filter-optionc">Option C</a></li>
		<li><a href="#" data-filter=".filter-optiond">Option D</a></li>

13 - CONTACT PAGE

Anthem comes with a fully working contact form. First open send-email.php file in a code editor and input your email address where you would like to receive emails. To set up the map, scroll to the bottom of the contact.html or contact-alt.html and set your longitude and latitude values for your location. It is ready to go.

14 - PROJECT PLANNER

Anthem comes with a fully working project planner form. Open projectplanner.php in a code editor. Change the $toemail value to your email where you would like to receive the planner requests. And you are good to go. If you would like to redirect users to a page other than thanks.html on planner submission. Change the $success_page_name value to the page or link you want to redirect users to. On successful form submission you will receive the planner request in a nice table layout to the email you specify. It is ready to go.

15 - SUBSCRIBE

Anthem also comes with a fully working subscribe form. Open subscribe.php in a code editor and input the email where you would like to receive the subscription email requests. Save it and you are ready to go.

16 - TWITTER

Anthem comes with a fully working twitter script. You will need your consumer key, consumer secret, user token and user secret. Which you can get from https://dev.twitter.com. Once you have these values, open index.php located in the twitter folder and input your keys and tokens. You will then need to open index.html or home-two.html, the two pages where we are using twitter, scroll to the bottom of the page to script section and input your twitter username. It is good to go.

Buttons

Anthem comes with two different buttons. Line button and fill button. Line button displays a border only button and on hover slides in a background color. Fill button displays a solid background color button and on hover displays a border only button. Using buttons is very easy as shown below.

	

<!--fill button-->
<a href="link" class="purple fill-button"><span>Link text</span></a>
<!-- change between background colors by substituting purple for other colors available such as white or black. -->


	

<!--line button-->
