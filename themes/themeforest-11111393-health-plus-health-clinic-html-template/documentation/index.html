File structure

  • assets/
    • css/
      • pages/...  (Style for each page)/ul>
      • core.css (Style for global components)
      • layout.css  (Main style for template)
      • vendor.css (Style for used library)
    • images/
    • js/
      • pages/...  (Script for each page)
      • layout.js (Script for template)
      • main.js (Script for global components)
      • other plugins scripts...
    • libs/  (included global plugins library)
    • vendors/  (included used plugins for this template)
  • HTML files

HTML Structure

Head Tag

<!DOCTYPE html>
<html lang="en">

<head>
    <title>Health Plus | Home Page</title>
    <meta http-equiv="content-type" content="text/html; charset=utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link type="text/css" rel="stylesheet" href="http://fonts.googleapis.com/css?family=Raleway:700,600,400,300,200">

    <link type="text/css" rel="stylesheet" href="assets/libs/ionicons/css/ionicons.min.css">
    <link type="text/css" rel="stylesheet" href="assets/vendors/medical-icons/style.css">

</head>

Slider

Bootstrap Carousel

We used Boostrap Carousel plugin for Home page banner.

For more information, please go on: http://getbootstrap.com/javascript/#carousel

Slider content

We configured that slider content is in this HTML markup:

 <div class="carousel-inner" role="listbox">
    <div class="item active">
      <img src="..." alt="...">
      <div class="carousel-caption">
        ...
      </div>
    </div>
    <!-- Add new slider follow below template -->
    <div class="item">
      <img src="..." alt="...">
      <div class="carousel-caption">
        ...
      </div>
    </div>
</div>

You can apply other configuration for this plugins, please visit https://github.com/woothemes/FlexSlider/wiki/FlexSlider-Properties to read more about this plugins API.
