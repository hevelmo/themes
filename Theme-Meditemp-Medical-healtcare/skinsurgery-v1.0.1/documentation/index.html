Setting the location for a map

If you want to change the location for a map on a page of the template, scroll to the bottom of the page and locate the line where the latlng variable is initialised in the javascript block. Change the Latitude and Longitude coordinates the the place you want to center the map on.

var latlng = new google.maps.LatLng(23.042817, 72.5672);
