Styles Styles customization and highlights

Heads up! As this theme relies heavily on the Bootstrap framework, the bootstrap.min.css file is essential for the theme to display correctly.

Stylesheets

Here's a list of the stylesheet files I'm using with this template:

File Name Description
style.css The main stylesheet for this template. This contains the styling for the all pages of the template.
bootstrap.min.css The stylesheet of the Bootstrap framework. This contains the css the create the grid system and other bootstrap elements. Please note that some of these styles may be overridden in the main.css file.
fontawesome.css This contains styling relavant to icon.
owltheme.css This contains styling for the owlcarousel.
owlcarousel.css This contains the styling for gallery sections.
owl.transitions.css This contains the styling for Owl Carousel CSS3 Transitions.

Setting the location for a map

If you want to change the location for a map on a page of the template, scroll to the bottom of the page and locate the line where the latlng variable is initialised in the javascript block. Change the Latitude and Longitude coordinates the the place you want to center the map on.

var latlng = new google.maps.LatLng(23.042817, 72.5672);
