ResumeX is a multipurpose, elegant and well crafted one page portfolio for creative professionals. Anyone can showcase their personal resume, work history, projects and other personal information beautifully using ResumeX. Though ResumeX is primarily developed for showcasing portfolio elements, however it can be creatively used to display other essential information for SME business and artistic people. Individual Section Structure

<div class="section-about" id="about-me">

    <div class="details">

        <div class="container">

            // Section content here

        </div>

    </div>

</div>

How to make any section?

For an example you want to create a section name Products then please go throung following instructions

STEP 01

Go to top nav section

 

STEP 02

Make a list element like 

<li><a href="#products" title="Products">Products</a></li>

STEP 03

Now you have to place section where you will put this section content. As example I have added link after skill section so go to bottom of skill section make a section like

<section id="products" class="products">

<h2 class="section-header">Products</h2>

<p class="section-intro ">

   Your text here<span>desert raisin</span> taro wakame pea sprouts. Broccoli rabe courgette rock <span>melon cress </span> epazotebell

</p>

<div class="container">

    <div class="row">

    <!-- You can use any column here as like as Bootstrap column stytem -->

        <div class="col-md-4 animated animate-from-left">

         Column

        </div>

        <div class="col-md-4">

            Column

        </div>

        <div class="col-md-4 animated animate-from-right">

            Column

            </div>

        </div>

    </div>

</div>

</section>
No need to add any link in nav if you want to create any section that will not a part on top navigation. And then no need to use id="" and no need to use appear class

How to add any animation in any element?

It is very easy just use just go to http://daneden.github.io/animate.css/ this site and choose an animation and add animation name by following way in any elemenet:

<h2 class="section-header wow slideInLeft">About Us</h2>

here "wow" dfine wow js it must be there and then "slideInLeft" define animate css effect.

Isotope  Dependent JS

<!-- Masonry Grid -->
<script src="js/jquery.isotope.js"></script>
 var $container = $('.blog-container'),
                colWidth = function () {
                    var w = $container.width(),
                        columnNum = 1,
                        columnWidth = 0;
                    if (w > 1200) {
                        columnNum = 3;
                    } else if (w > 900) {
                        columnNum = 3;
                    } else if (w > 600) {
                        columnNum = 3;
                    } else if (w > 300) {
                        columnNum = 1;
                    }
                    columnWidth = Math.floor(w / columnNum);
                    $container.find('.item').each(function () {
                        var $item = $(this),
                            multiplier_w = $item.attr('class').match(/item-w(\d)/),
                            multiplier_h = $item.attr('class').match(/item-h(\d)/),
                            width = multiplier_w ? columnWidth * multiplier_w[1] - 20 : columnWidth - 20;
//                                height = multiplier_h ? columnWidth*multiplier_h[1]*0.5-4 : columnWidth*0.5-4;
                        $item.css({
                            width: width
                            //                            height: height
                        });
                    });
                    return columnWidth;
                },
                isotope = function () {
                    $container.isotope({
                        resizable: false,
                        itemSelector: '.item',
                        masonry: {
                            columnWidth: colWidth()
                            //                            gutterWidth: 10
                        }
                    });
                };
            isotope();
            $(window).smartresize(isotope);
            $(window).load(isotope);

 

 

Themes And Layout

This template have 5 predefine themes. All themes files have in css/themes folder. This template have following themes.

If you want to use any theme you have to just replace theme file in following line:

<link href="css/default-theme.css" type="text/css" rel="stylesheet">
Now it is default theme. If you want to use blue theme then just replace path by following way. Your theme will be change instantly.
<link href="css/themes/blue-theme.css" type="text/css" rel="stylesheet">
